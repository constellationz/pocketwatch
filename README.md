# pocketwatch

A MERN stack time tracking app

## Getting Started

1. Clone the git repository

```bash
git clone https://github.com/constellationz/pocketwatch && cd pocketwatch
```

2. Install dependencies

```bash
npm install
```

3. Set up `.env`

```
NODE_ENV = development
PORT = 5000
MONGO_URI = <your mongo cluster URI>
JWT_SECRET = abc123
```

4. Host test server

```bash
npm run nodemon
```
